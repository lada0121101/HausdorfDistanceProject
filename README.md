# HausdorfDistanceProject


Класс K2Tree представляет K2 дерево как граф(представление на связных списках)
minx miny maxx maxy представляют иднексы квадрата в матрице, которую представляет узел
value - значение в узле (0 или 1)
childs - дети узла

Класс K2TreeMatrix представляет собой матрицу, которую копактно ужимет K2 дерево

BitArrayExtension это класс обертка над BitArray, которая позволяет делать двойную индекацию к одномерному массиву, тем самы превращая одномерный массив в матрицу

В классе Convertor представленны два алгоритма 1. трансформации набора точек в матрицу 2. Создание из матрицы K2 дерева, компактно представляющего данную матрицу
